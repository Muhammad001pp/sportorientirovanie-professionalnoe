Как пользоваться

Остановить все:
npm run dev:mac:stop
Запустить все:
npm run dev:mac
Перезапустить все:
npm run dev:mac:restart
Альтернатива (без вкладок, в одном терминале):
npm run dev:all
Замечание

Если ADMIN_KEY или NEXT_PUBLIC_CONVEX_URL не заданы, скрипт подскажет это в консоли (для admin/.env.local).
Если уже запущены старые экземпляры, restart корректно их убьёт и запустит новые.
